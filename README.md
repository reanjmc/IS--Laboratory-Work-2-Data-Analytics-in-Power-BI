**GUIDE QUESTIONS**

__1. Why is removing duplicates important in data analysis?__
Removing duplicates is important because it prevents the same data from being counted more than once, which can affect the accuracy of the analysis.

__2. What problems can missing values cause in reports?__
Missing values can cause incorrect results in calculations and may lead to incomplete or misleading reports.

__3. How does correct data typing affect calculations?__
Correct data typing ensures that numbers, dates, and text are processed properly so calculations and analysis work correctly.

__4. What naming conventions improve data readability?__
Using clear and consistent names such as lowercase letters and underscores (like order_id or total_sales) makes the data easier to read and understand.
