**GUIDE QUESTIONS**

__1. Why is removing duplicates important in data analysis?__

Removing duplicates ensures that data is accurate and prevents double counting in reports.

__2. What problems can missing values cause in reports?__

Missing values can lead to incorrect calculations and incomplete analysis.

__3. How does correct data typing affect calculations?__

Correct data types ensure calculations, sorting, and filtering work properly.

__4. What naming conventions improve data readability?__

Using clear and consistent names like lowercase with underscores improves readability.
