# Power Query – Data Cleaning - GUIDE QUESTIONS

__1. Why is removing duplicates important in data analysis?__

Removing duplicates ensures that data is accurate and prevents double counting in reports.

__2. What problems can missing values cause in reports?__

Missing values can lead to incorrect calculations and incomplete analysis.

__3. How does correct data typing affect calculations?__

Correct data types ensure calculations, sorting, and filtering work properly.

__4. What naming conventions improve data readability?__

Using clear and consistent names like lowercase with underscores improves readability.


**Merge and Append Queries - GUIDE QUESTIONS**

__1. What is the difference between Merge and Append?__

Merge combines tables horizontally using a common column, while Append stacks tables vertically.

__2. Why use Inner Join instead of Left Join?__

Inner Join returns only records that exist in both tables, ensuring matched data.

__3. What happens to unmatched records in an Inner Join?__

Unmatched records are removed from the result.

__4. When is Append more appropriate than Merge?__

Append is used when combining multiple tables with the same structure, such as monthly sales data.
